# HR-Analytics-Dashboard of IBM Employee Attrition
Power BI dashboard project using SQL and analytics insights

## Project Overview
This project focuses on analyzing employee data to identify workforce trends, attrition patterns, and business insights using Power BI and SQL.

## Tools Used
- Power BI
- SQL
- Excel / CSV
- Data Analytics

## Key Insights
- Employee attrition analysis
- Department-wise performance tracking
- KPI monitoring
- Trend analysis and visualization

## Dashboard Features
- Interactive filters
- KPI cards
- Trend charts
- Department analysis
- Employee insights

## Files Included
- Power BI Dashboard (.pbix)
- Dataset (.csv)
- SQL Queries (.sql)
- Dashboard Screenshots

## Business Impact
This dashboard helps organizations make data-driven HR decisions by improving workforce analysis and identifying key attrition factors.
